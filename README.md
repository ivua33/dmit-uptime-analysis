**Q：DMIT 支持退款吗？**

支持有条件退款。3 天内使用流量不超过 30GB 可以全额退，30 天内可按剩余价值退（会扣除支付手续费）。下单前确认好自己的需求，因为条件比较具体，不要抱着"随便试试不合适再退"的想法进来。

**Q：工单回复要等多久？**

非紧急工单（非托管服务）官方注明最长 72 小时，实际用下来大多数时候 6 小时内有回音，紧急的情况几十分钟也有。中文支持有，但不是每次都能保证，英文沟通更稳妥。
